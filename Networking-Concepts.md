# 🐧 Lab: Linux Shells e Terminais

**Plataforma:** TryHackMe
**Objetivo:** Explorar os diferentes tipos de shells no ecossistema Linux e a interação direta com o sistema operativo.

### 📝 Atividades Realizadas:
- **Interação com a Shell:** Prática de comandos para manipulação de ficheiros e permissões.
- **Ambiente de Terminal:** Compreensão de como a shell atua como intermediária entre o utilizador e o kernel do sistema.

### 💡 Por que isto é importante:
O Linux é a base da maioria das infraestruturas de segurança e servidores web. Ter agilidade no terminal permite uma resposta muito mais rápida a incidentes e facilita a automação de tarefas de defesa.
