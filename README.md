## 🛡️ Meu Primeiro Projeto: Análise de Redes e Protocolos
**Laboratório:** Fundamentos de Redes (TryHackMe)

### 📝 O que foi realizado:
Neste laboratório, explorei como os dados viajam pela internet. Utilizei ferramentas para identificar portas abertas e entender o tráfego de rede, focando em como proteger essas comunicações contra acessos não autorizados.

### 🛠️ Ferramentas e Conceitos:
- **Wireshark:** Observação de pacotes em tempo real.
- **Protocolos:** Diferença prática entre TCP (segurança na entrega) e UDP (velocidade).
- **Terminal Linux:** Comandos essenciais de diagnóstico de rede.

> Este projeto demonstra minha base técnica em infraestrutura, essencial para operações de segurança (SOC).
